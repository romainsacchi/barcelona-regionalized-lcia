# Course Data

This folder contains bundled data used during the course.

Included file:
- `lci-bafu.xlsx`: the Brightway-compatible Excel version of the BAFU:2025 inventory used in Day 1.

Notes:
- Participants do not need to download the BAFU database before the course.
- The BAFU website link elsewhere in the repository is provided for reference and terms of use.
- `ecoinvent` is not included in this repository and is not used in the hands-on course because of licensing constraints.
